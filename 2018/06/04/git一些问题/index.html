原因

（来自某网友的回答）如果手动创建git repo的话，默认是没有remote的，需要手动添加。git remote add origin . “origin”可以换成其它名字，url是远程仓库的地址（可以是本地硬盘上的某个git repo）。如果用git clone的话，会自动创建一个remote origin
解决方法：git remote add origin （远程仓库地址）
这条命令会在最上面那个文件夹里添加remotes这个文件，这样就可以git pull
